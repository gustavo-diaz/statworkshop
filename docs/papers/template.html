Abstract
This is a template to write an academic paper in Quarto. In this space, you normally write the abstract, which is a summary of the paper in no more than 120-200 words.

1 Introduction

Papers usually start with an introduction. See this link for some tips on how to write introductions.7

2 Code

3 Citations

Citations by default are in Chicago style. For example, you can see Wickham (2016b) to learn more about working with R and RStudio. See here for more information on how to use other citation styles.

4 Figures

4.1 From R

Figure 1 shows how to plot a figure from R code. See Wickham (2016a) for more on how to make pretty plots.8

4.2 From external sources

Sometimes you may want to include a figure that was not made in R. Alternatively, when your data analysis becomes too cumbersome, you may want to make your figures in a separate file, and then import them. Figure 2 shows an example using a URL. In practice, you would use the path of file in your working directory.

5 Tables

5.1 Pipe tables

You can make tables “by hand” using pipe tables. Like Table 1 shows.

You can even use a Tables Generator to make them interactively and then copy and paste in your .qmd file.

5.2 From R

Of course, you can also make tables in R and then print them in a nice format. I like to use the tinytable for simple tables.9

For example, Table 2 is made “by hand” with R code. Of course, you can also plot tidy data objects, like Table 3 shows.

6 Conclusion

You can use this template to start writing your paper, then make it your own by customizing it in any way you would like. You can use a journal-style template or make your own.
